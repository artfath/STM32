Release Notes for

STM32Cube Expansion of Class B library

Copyright 2017 STMicroelectronics

 

Update History

V2.2.0 / 30-September-2017

Main Changes

  • The stack update after implementation of the updated HAL, CMSIS and BSP drivers supporting the latest products and covering known limitations. The stack was re-tested for newly added STM32 family members.
  • Re-tested for the latest versions of IAR & KEIL & Ac6 compilers.
  • IWDG and/or WWDG refresh is applied inside FailSafePOR() subroutine when debug parameter NO_RESET_AT_FAIL_MODE is applied and some of the watchdogs are actively used.
  • Constant area stored at RAM by compiler initializer is refreshed after return from variable memory start up self-test for IAR and Ac6 (not possible for KEIL).
  • Casting of some constants is added to comply with MISRA 2004 checks.
  • Duplicate call of STL_InitClock_Xcross_Measurement() function was removed from STL_InitRunTimeChecks() procedure.
  • The version applied for UL re-certification process.

V2.1.0 / 19-December-2016

Main Changes

  • The stack update after implementation of the updated HAL, CMSIS and BSP drivers supporting the latest products and covering known limitations and after testing all the associated GCC projects supporting Ac6 compiler.
  • Re-tested for the latest versions of IAR & KEIL compilers.
  • The version applied for UL re-certification process

V2.0.2 / 9-May-2016

Main Changes

  • This package was tested for GCC compiler only
  • HAL layer is updated to V1.4.4 where limitation of LCD driver was corrected.
  • Minor changes related to debugging features (change of UART handle structure, definition of putchar() function modified for GCC, startup copy handler restored after SRAM destructive startup test for GCC)
  • Formal change of comments at middleware source files (spelling correction)

V2.0.1 / 25-April-2016

Main Changes

  • First preliminary release of the self-test library (STL) tested for GCC compiler (derived from version 2.0.0 certified by UL certification authority for IAR and KEIL compiler)

V2.0.0 / 14-December-2015

Main Changes

  • First official release of the FW certified by UL certification authority
  • License disclaimer update
  • Correction after MISRA check
  • Use of latest HAL drivers

V2.0.0RC5 / 11-November-2015

Main Changes

  • HSI option is implemented additionally
  • IWDG is started prior its initialization

V2.0.0RC4 / 17-August-2015

Main Changes

  • RAM tests are replaced by procedures written in assembler
  • IAR CRC calculation issue is compensated (HAL driver is replaced by direct control of CRC unit)

V2.0.0RC2 / 8-December-2014

Main Changes

  • Added support of STM32L1xx HAL drivers

V2.0.0RC1 / 11-June-2014

Main Changes

  • First release of common self-test stack supported by HAL drivers
